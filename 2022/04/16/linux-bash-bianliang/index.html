1.1.bash 变量

  • 什么是变量

    变量是计算内存的单元，其中存放的值可以改变，当Shell脚本需要保存一些信息时，如一个文件名或是一个数字，就把它存放在一个变量中。每个变量有一个名字，所以很容易引用它。使用变量可以保存有用的信息，是系统获知用户相关设置，变量也可以用于保存暂时信息。

  • 变量分类

    • 用户自定义变量
    • 环境变量
    • 位置参数变量
    • 预定义变量

1.2.变量命名规则

  • 由字母、数字和下划线组成，不能以数字开头
  • 变量默认是字符串类型，如果要需要其他类型，需要指定

1.3.设置变量

  • 变量和值之间由等号连接，等号两侧不能有空格，如果变量的值有空格，需要用双或单引号引住
  • 变量中的值可以用 \ 转义
  • 变量叠加
    • 对变量值叠加，变量需要 ”$变量名”或${变量名}引用
    • name=lz ; 叠加：name=”$name”p
    • name=lz ；叠加： name=${name}p
    • 结果： neme=lzp
  • 把命令的值作为变量值赋予给变量，则命令需要用反引号或$()包含命令；
  • 建议将环境变量大写；

2.用户自定义变量

  • 变量定义
    • name=lzp
  • 变量引用
    • echo $name
  • 变量查看
    • set
  • 变量删除
    • unset 变量名

3.环境变量

变量中主要保存的是和系统操作环境相关的数据

用户变量只在当前shell中执行，环境变量在当前shell及其子shell都可执行，如果将环境变量写入配置文件中，则全部shell均可使用

  1. 父shell与子shell
进入命令界面，即进入标准bash shell 
[root@~]#bash 输入bash,代表在当前shell中，再进入一个shell
[root@~]#csh 再次进入
[root@~]#yum install psmisc; 下载pstree命令包
[root@~]#pstree 查询
结果：login-bash-bash-csh-pstree
则刚进入的界面为父shell,每个接下来的shell,均是传承关系；
[root@~]:exit ：退出最后一个shell
  1. 定义环境变量
  • export 变量=变量值
  • env ；查询环境变量
  • unset 变量名 ；删除变量
  1. 系统常见环境变量
  • $PATH
    • PATH=/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/root/bin
    • PATH存放的是系统查找命令的路径，输入ls命令，系统会在PATH变量中的路径中依次搜索；
    • PATH=”$PATH”:/root (PATH变量叠加)
  1. 环境变量配置文件

  • 写入环境变量配置文件中的变量才会永久生效

  • source命令

    • source 配置文件 或 .配置文件 都会让配置文件不用重启就能生效

  • 常见的环境变量配置文件

    • 学完shell编程之后，再来学习配置文件内具体内容

4.位置参数变量

  • 相当于形参的作用，在调用脚本的时候输入变量值，传到脚本之中
#脚本文件：
#!/#!/bin/bash
echo $0  #代表命令本身
echo $1	 #代表第一个参数
echo $2	 #代表第二个参数

执行脚本：
[root@~]# chomd 755 /sh/test.sh hello world //传入两个参数： hello 和 world
[root@~]# ./test.sh

执行结果:
./test.sh
hello
world
  • $@和$#
#!/bin/bash

	for i in "$*"
		do
			echo $i
		done

	for y in "$@"
		do 
			echo $y
		done
输入命令： 
[root@~]./test.sh hello world
执行结果：
hello world
hello
world

5.预定义变量

  • 命令执行的 || 和 && 正确与否，就是以$?命令判定的

6.接受键盘输入

  • read [选项] [变量名]
    • -p “提示消息” ：在等待read输入时，输出提示信息
    • -t 秒数 ：指定输入的等待时间
    • -n 字符数：read命令就会只接受n个字符
    • -s : 隐藏输入的数据，适于机密信息的输入
#!/bin/bash

read -t 20 -p "please input your name " name  #将键盘键入的变量赋值给name

echo "Name is $name"

read -t 10 -p "please input your age " -s age #将键盘键入的变量赋值给age

echo "Age is $age"

7.数值运算和运算符

  • 声明变量类型

    • declare [+/-] [选项] 变量名

      -：给变量设定类型属性

      +：取消变量的类型属性

      -i：将变量声明为整数型

      -x：将变量声明为环境变量

      -p：显示指定变量的被声明的类型

  • 数值运算

    • 方法一: 将代表结果的变量定义为数值类型

      [root@~]#a=1
[root@~]#b=2
[root@~]#declare -i c=$a+$b
